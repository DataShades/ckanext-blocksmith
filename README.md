# ckanext-blocksmith

Blocksmith is a CKAN extension that allows you to create and manage pages using a visual editor.

## Requirements

CKAN 2.10+
Python 3.9+

## Installation

To install ckanext-blocksmith:
    ```
    pip install -e .
    ```
